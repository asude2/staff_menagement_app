# 🚀 Modern User Management Dashboard

A professional, high-performance **User Management System** built with **React 18** and **Tailwind CSS v4**. This project is designed to organize customer or personnel records with real-time data visualization and persistent storage.

## ✨ Features

- **Full CRUD Operations**: Create, Read, Update, and Delete user records seamlessly.
- **Data Persistence**: Uses **LocalStorage** to keep your data safe even after a page refresh.
- **Interactive Dashboard**:
  - **Live Statistics**: Real-time tracking of total users and local records.
- **Advanced UX**:
  - **Live Search**: Instant filtering by Name, Email, or Company.
  - **Action Confirmations**: Guardrails for deleting records.
  - **Toast Notifications**: Modern alerts for every successful action.
- **Responsive Design**: Fully compatible with mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

- **Framework**: [React 18 (Vite)](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Deployment**: [Netlify](https://asude-kalgi-web-proje.netlify.app/)

⚙️ Installation & Setup

1. Clone the repository:

2. Install dependencies:
    npm install

3. Start the development server:
    npm run dev
